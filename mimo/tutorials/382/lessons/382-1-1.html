As the content of these sections will have quite a few things in common, it's smart to use another set of &lt;div&gt; elements with the same class.

Within each section, add a &lt;div&gt; element with a class attribute set to container.

...
&lt;body&gt;
&lt;div id="header"&gt;
  &lt;div class="container"&gt;
  &lt;/div&gt;
&lt;/div&gt;
&lt;div id="content"&gt;
  &lt;div class="container"&gt;
  &lt;/div&gt;
&lt;/div&gt;
&lt;div id="footer"&gt;
  &lt;div class="container"&gt;
  &lt;/div&gt;
&lt;/div&gt;
&lt;/body&gt;
...
