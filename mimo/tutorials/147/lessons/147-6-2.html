In short, CSS is a language that describes the presentation of a web page.

Whenever we change the style of an HTML element, we use a property name, a colon, a value and a semicolon.

<!doctype html>
<html>
 <body>
  <p style=" color :red; color = red">I'm simply red.</p>
 </body>
</html>
<html><body><p style="color:red;">I'm simply red.</p></body></html>
]]>

Sweet! With this simple declaration, we've changed the paragraph's font color to red.

Didn't we say that we use a property name, a colon, a value and a semicolon?
