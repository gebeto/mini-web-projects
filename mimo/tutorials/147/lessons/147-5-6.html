With the rowspan attribute, header and table cells can span across multiple rows. What might be the equivalent attribute for the number of columns?

<!doctype html>
<html>
 <body>
  <table>
   <tr>
    <th  colspan rowspan="2">Name</th>
   </tr>
   <tr>
    <td>Henry</td>
    <td>Ameseder</td>
   </tr>
  </table>
 </body>
</html>
<html>
<body><table><tr><th colspan="2">Name</th></tr><tr><td>Henry</td><td>Ameseder</td></tr></table></body>
</html>
]]>

Nice! By setting the colspan attribute to "2", the header cell spans across the two cells below it.

If rowspan is the attribute for spanning across rows, there's probably a different attribute for columns.
