While we're at it, what if we want to use a font with a name that consists of multiple words?

<!doctype html>
<html>
 <body>
  <p style="font-family: 'times new roman' "times new roman" times new roman;">I'm still in between tags.</p>
 </body>
</html>
<html><body><p style="font-family:'times new roman';">I'm still in between tags.</p></body></html>
]]>

Fantastic! As we can't use double quotation marks, we have to fall back to single quotation marks.

We have to use quotation marks, but we can't use double quotation marks, as they'd mark the end of the style attribute's value.
