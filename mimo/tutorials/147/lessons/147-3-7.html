A great way to include videos in a web page is to use YouTube and the easiest way to get there is to add an inline frame.

<!doctype html>
<html>
 <body>
  <iframe width="420" height="315"  src href="https://www.youtube.com/embed/Vhh_GeBPOhs">
  </iframe>
 </body>
</html>
<html>
<body><iframe src="https://www.youtube.com/embed/Vhh_GeBPOhs"></iframe></body></html>
]]>

Fantastic! The <iframe> tag has very similar attributes as the <img> tag, which makes it really easy to use.

With the exception of a closing tag, the <iframe> tag is very similar to the <img> tag.
