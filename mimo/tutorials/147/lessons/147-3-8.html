If, however, we actually have the web address of an actual movie clip or a video stream, we can use the <video> tag to add it to the web page.

<!doctype html>
<html>
 <body>
  <video src="https://getmimo.com/r/video.mov"  controls controls=""></video>
 </body>
</html>
<html>
<body><video src="https://getmimo.com/r/video.mov" control></video></body>
</html>
]]>

Sweet! If we include the controls attribute, the web browser displays buttons to play and pause the video.

Note that controls is a boolean attribute, which means it doesn't need a value. If it's there, it's treated as true; otherwise, it's treated as false.
