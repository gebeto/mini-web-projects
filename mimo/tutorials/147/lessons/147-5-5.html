This time, let's add a header row to this table and find out what the cells mean!

<!doctype html>
<html>
 <body>
  <table>
   <tr>
     <th>Month</th> <td>Month</td> 
    <th>Savings</th>
   </tr>
   <tr>
    <td>January</td>
    <td>$150</td>
   </tr>
   <tr>
    <td>February</td>
    <td>$150</td>
   </tr>
  </table>
 </body>
</html>
<html>
<body><table><tr><th>Month</th><th>Savings</th></tr><tr><td>January</td><td>$150</td></tr><tr><td>February</td><td>$150</td></tr></table></body>
</html>
]]>

Good stuff! The <th> tag defines a header cell. If we make every cell in a row a header cell, they form a header row.
