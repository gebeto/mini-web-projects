There's a tag at the beginning of a webpage that we've neglected so far: the doctype, which tells the web browser about the version of HTML we're using.

Can you set the doctype to HTML5, the current version of HTML?

<!doctype html xml>
<html>
 <head>
  <title>My web page</title>
 </head>
 <body>
  Hello!
 </body>
</html>
<html>
<head>
<title>My web page</title>
</head>
<body>Hello!</body>
</html>
]]>

Nice! <!doctype html> means that we're using HTML5. If we skip the doctype, the browser has to guess, which is never a great idea.
