Let's change the web page's background color to gray!

<!doctype html>
<html>
 <body style=" background-color :gray; gray">
  I come in many shades of gray.
 </body>
</html>
<html><body style="background-color:gray;">I'm still in between tags.</body></html>
]]>

Sweet! The value of the style attribute is a style declaration written in CSS, which stands for Cascading Style Sheets.

It's a little more complicated, as the value of the style attribute isn't an HTML value but a CSS style declaration.
