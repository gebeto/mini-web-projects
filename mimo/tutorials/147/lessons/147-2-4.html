In the last example, we've used a URL, the absolute address of a web page. This time, let's create a local link!

<!doctype html>
<html lang="en-US">
 <body>
  <a href=" index .html https://getmimo.com">
   Take me somewhere!
  </a>
 </body>
</html>
<html lang="en-US">
<body><a href="https://getmimo.com/index.html">Take me somewhere!</a></body>
</html>
]]>

Sweet! Local links direct us to web pages on the same computer or web server as the original web page's.

That kind of looks like another URL, doesn't it?
