By grouping block-level HTML elements, the <div> tag creates sections of a web page that we can format with CSS.

<!doctype html>
<html>
 <body>
  <div style="width:200px;">
   <h1>A familiar story</h1>
   <p>A long time ago in a galaxy far, far away ...</p>
   </div> <div>
 </body>
</html>
<html><body><div style="width:200px;"><h1>A familiar story</h1>
<p>A long time ago in a galaxy far, far away ...</p></div></body></html>
]]>
