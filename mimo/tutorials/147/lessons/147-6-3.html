How about we also change the paragraph's font to something more appealing?

<!doctype html>
<html>
 <body>
  <p style="color:red; font-family background-color:arial;">I'm still in between tags.</p>
 </body>
</html>
<html><body><p style="color:red;font-family:arial;">I'm still in between tags.</p></body></html>
]]>

No surprises: the font-family property defines an HTML element's font. What's more interesting is that it happens within the very same attribute.

Didn't we want to change the font and not the background color?
