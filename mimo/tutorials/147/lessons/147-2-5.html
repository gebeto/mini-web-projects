The last attribute we'll look at is the target, which we can use to open a web page in a new tab or window.

<!doctype html>
<html lang="en-US">
 <body>
  <a href="https://getmimo.com" target=" _blank _self">
   Take me somewhere!
  </a>
 </body>
</html>
<html lang="en-US">
<body><a href="https://getmimo.com" target="_blank">Take me somewhere!</a></body>
</html>
]]>

Sweet! Using _blank, if we click on the link, the linked web page opens in a new tab or window.

While both of these values work, _self opens the linked web page in the same tab or window.
