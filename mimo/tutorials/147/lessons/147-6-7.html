As a rule, HTML elements are either block-level or inline elements.

While block-level elements like <h1> and <p> are preceded and followed by a line break, inline elements like <em> and <strong> aren't.

<!doctype html>
<html>
 <body>
  <h1>Block-level element</h1>
  <p>
    <em>Inline</em> <p>Inline</p> element
  </p>
 </body>
</html>
<html><body><h1>Block-level element</h1><p><em>Inline</em> element</p></body></html>
]]>

Nice! As we already have a block-level element, we use <em> to create an inline element without adding additional line breaks.

Like <h1>, <p> is a block-level element, which is why it wouldn't make that much sense to add another paragraph, would it?
