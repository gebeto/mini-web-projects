Last but not least, we can also add a border to a table.

<!doctype html>
<html>
 <body>
  <table border=" 1 2">
   <tr>
    <th colspan="2">Name</th>
   </tr>
   <tr>
    <td>Henry</td>
    <td>Ameseder</td>
   </tr>
  </table>
 </body>
</html>
<html>
<body><table border="1"><tr><th colspan="2">Name</th></tr><tr><td>Henry</td><td>Ameseder</td></tr></table></body>
</html>
]]>

Great! The border attribute belongs in the <table> tag. With the values 1 and 0, we can define whether or not the table should have a border.

We can only use the border attribute to define whether or not the table should have a border; we can't use it to set a border width.
