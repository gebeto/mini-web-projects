Let's talk about indentation—the spaces at the beginning of a line. When we add a nested tag in a new line, we indent it a bit more than the surrounding tag.

Can you add a <p> tag with indentation?

<!doctype html>
<html>
 <body>
  <h1>Welcome!</h1>
   <p> </p>  
   HTML is not a <a href="https://en.wikipedia.org/wiki/Programming_language">programming language</a>
  </p>
 </body>
</html>
<html>
<body><h1>Welcome!</h1><p>HTML is not a <a href="https://en.wikipedia.org/wiki/Programming_language">programming language</a></p></body>
</html>
]]>

Swee-heet! Indentation is a great way to make a web page's HTML code more readable.
