Because whitespace in a paragraph is also treated as a single space, we need to find another way to add line breaks.

<!doctype html>
<html>
 <body>
  <h1>My favorite things</h1>
  <p>
   Raindrops on roses <br> <p> </p>
   Whiskers on kittens
  </p>
 </body>
</html>
<html>
<body><h1>My favorite things</h1><p>Raindrops on roses<br>Whiskers on kittens</p></body>
</html>
]]>

What a great idea! We can use the <br> tag, which inserts a line break.
