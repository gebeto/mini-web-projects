Then, how can we get more than a single column?

<!doctype html>
<html>
 <body>
  <table>
   <tr>
    <td>January</td>
    <td>$150</td>
   </tr>
   <tr>
    <td>February</td>
     <td>$150</td> <tr>$150</tr>
   </tr>
  </table>
 </body>
</html>
<html>
<body><table><tr><td>January</td><td>$150</td></tr><tr><td>February</td><td>$150</td></tr></table></body>
</html>
]]>

Sweet! By adding another cell to the table row, the table gets a second column.

Adding table rows isn't always the answer.
