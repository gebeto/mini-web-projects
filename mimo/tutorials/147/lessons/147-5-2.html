The truth is that we don't really create columns. Instead, we add cells, which the table counts as the number of columns.

<!doctype html>
<html>
 <body>
  <table>
   <tr>
     <td>January</td> <tr>January</tr> 
   </tr>
   <tr>
    <td>February</td>
   </tr>
  </table>
 </body>
</html>
<html>
<body><table><tr><td>January</td></tr><tr><td>February</td></tr></table></body>
</html>
]]>

Sweet, we have two rows and a column.
