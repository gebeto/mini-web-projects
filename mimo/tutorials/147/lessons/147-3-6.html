We can use so-called inline frames to display a separate web page within a web page.

The <iframe> tag is pretty similar to the <img> tag, except that it has a closing tag.

<!doctype html>
<html>
 <body>
   <iframe></iframe> <iframe>
 </body>
</html>
<html>
<body><iframe></iframe></body>
</html>
]]>

Sweet! Even though it doesn't look like it, with the <iframe> tag, we can do lots of fun things.
