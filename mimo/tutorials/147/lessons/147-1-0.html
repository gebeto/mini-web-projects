A very important thing to notice is that HTML doesn’t care too much about whitespace.

<!doctype html>
<html>
 <body>
  My favorite things
  Raindrops on roses
  Whiskers on kittens
  Bright copper kettles
  Warm woolen mittens
 </body>
</html>
<html>
<body>My favorite things Raindrops on roses Whiskers on kittens Bright copper kettles Warm woolen mittens</body>
</html>
]]>

See that? It doesn’t make a difference if we add multiple spaces or line breaks. HTML interprets them as whitespace and displays a single space.
