Also, kind of like an essay, we can divide a web page into paragraphs.

<!doctype html>
<html>
 <body>
  <h1>My favorite things</h1>
  <p>
   Raindrops on roses
   Whiskers on kittens
   Bright copper kettles
   Warm woolen mittens
   </p> </h1>
 </body>
</html>
<html>
<body><h1>My favorite things</h1><p>Raindrops on roses Whiskers on kittens Bright copper kettles Warm woolen mittens</p></body>
</html>
]]>

Sweet! The <p> tag defines a paragraph, to which browsers automatically add space above and below.

HTML rule #1: there are no rules.
HTML rule #2: close tags in the same order you open them.
