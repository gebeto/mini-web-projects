Now we know what a class is, what states and behaviors are for and how to create classes and instances. That's really great!

class Sandwich
 attr_reader :filling
 def initialize
  @filling = []
 end
 def add(filling)
  @filling.push(filling)
 end
end

sandwich = Sandwich.new
sandwich.add("cheese")
sandwich.add("")
sandwich.filling
["cheese", ""]
