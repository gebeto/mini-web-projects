Explorers don't just have names, have they? Luckily, we can add as many states to classes as we want.

class Explorer
 attr_reader :name, :age, :origin
 def initialize(name, age, origin)
  @name = name
  @age = age
  @origin = origin
 end
end

explorer = Explorer.new("Bob", 30, "Atlantis")
explorer. name age origin 
Bob 30 Atlantis

See that? The person variable is an instance of Explorer, which means it can hold loads of goodies for us to use.
