Create a Discovery module with a method. Include it in the Explorer class and call the method on bob, an instance of the class.

module Discovery
 def jig
  puts "#{@name} does a jig!"
  puts "We mastered Ruby!"
 end
end
class Explorer
 include Discovery
 attr_reader :name
 
 def initialize(name)
  @name = name
 end
end

bob = Explorer.new("Bob")
bob.jig
