So, we created an Explorer class and gave it a state, the name attribute. These attributes are stored in so-called in instance variables.

class Explorer
 attr_reader :name
 def initialize(name)
  @name :name name = name
 end
end

explorer = Explorer.new("Bob")
print explorer.name Bob

That's it! We create instance variables with a @ sign and access them with a . sign through an instance of the class.

Psst: states don't do anything, they just store information.
