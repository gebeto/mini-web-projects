Alright, ready for another one? Let's check out the logical or, or ||.

These odd-looking symbols are called pipes. Much like with &amp;&amp;, Ruby will look for comparisons on both sides of the operator.

print 2 < 3 ||  3 == 3 3 != 3  true
 true

See that? Only one of the values we compare has to be true for the || to evaluate to true.
