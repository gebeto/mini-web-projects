Now, let's create a Film class with all the attributes and methods that belong to a movie!

class Film:
  def __init__(self, title, year, actors):
    self.title = title
    self.year = year
    self.actors = actors
  def display_details(self):
    title = self.title
    actors = str(self.actors)
    print(title + " with: " + actors)

film = Film("Holy Grail", 1975, ["J. Cleese", "T. Gilliam"])
film.display_details()
Holy Grail with:['J.Cleese', 'T.Gilliam']

Great! Finally, we can collect these attributes and methods in a single entity - a class.
