Variables are containers that help us store and access information. We create a variable by writing down a name, an = sign and a value.

variable1 = "They formed in 1969"
variable2 = " in England"
print(variable1 +  variable2 " in England" )
They formed in 1969 in England

Great! We can use a variable's name to access the value it contains.
