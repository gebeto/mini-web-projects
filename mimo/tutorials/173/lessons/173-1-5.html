A variable changes its type when we give it a value of another type.

Let's change the value of the containervariable to something that isn't a string, then!

container = "Arthur"
print(type(container))
container = 1969 3.14159265359 "Patsy"
print(type(container))
<class 'str'>                                                   <class 'int'> <class 'str'>                                                     <class 'float'>

Wow, and that's still the same variable! We've changed the type of the container variable by giving it a value of a different type.
