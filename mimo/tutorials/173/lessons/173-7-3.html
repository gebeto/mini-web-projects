Sometimes we want a custom function to return a value. Well, there's a keyword for that.

def to_fahrenheit(celsius): 
  f = celsius * 9 / 5 + 32
  return f
        
degrees = to_fahrenheit( 100 23 0 )
print(str(degrees) + " degrees")
212.0 degrees 73.4 degrees 32.0 degrees

See that? By putting a value or variable after the return keyword, we return a value that we can store in a variable or reuse in a different function.
