We can also go through, or iterate over, strings. Let's use a loop to check if a sentence contains a certain character!

text = "One ring to rule them all"
found = False
for character in text:
  if character ==  "e" "z" text:
    found = True
print("Found: " + str(found))
Found: True Found: False

Great! As you can see, our for loop iterates through each letter in text and sets found equal to true if it finds the letter we have chosen.
