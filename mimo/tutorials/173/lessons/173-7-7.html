That's not the end of the story, though. Functions can't just call other functions but also themselves.

def factorial(n):
  if n == 1:
    return 1
  else:
    return (n * factorial(n - 1))

number = factorial( 1 5 10 )
print(number)
1 120 3628800

See that? This recursive function multiplies n with the factorial of n - 1, until n equals 1.

Psst: a factorial of a number is the product of all the numbers from 1 to that number.
