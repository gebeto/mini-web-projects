We create a word, or string, whenever we enclose a value in quotation marks.

name = " Arthur 42 1.85 "
hometown = 'Camelot'
print(type(name))
print(type(hometown))
<class 'str'>
<class 'str'> <class 'str'>
<class 'str'> <class 'str'>
<class 'str'>

See that? As the type() function proves, everything we put in quotation marks becomes a string, including numeric values.

Psst: we use either single or double quotes for str-type values.
