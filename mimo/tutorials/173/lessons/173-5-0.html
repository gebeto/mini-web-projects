A list stores multiple values of any type and uses indexes to distinguish these values.

myList = ["pam", "joe", 23, 4]
print( myList[0] myList[1] myList[2])
pam joe 23

In this example, the first line sets the list and its values. Then, the second line accesses individual values by using an index number inside brackets.

Psst: notice how the list allows us to include both strings and numbers? Yup, we're not restrained by type.
