We can group together values into so-called tuples using parentheses and separating them with commas.

person = ("Arthur", 1969)
print(person[ 0 1 ])
Arthur 1969

See that? Tuples group any number of values into a single variable. We access these values with brackets and an index that starts at 0.

Psst: once created, we can't change the values, or elements, of a tuple.
