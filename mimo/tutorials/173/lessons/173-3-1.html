Boolean values are great because they allow us to see how values compare to each other.

Let's give it a shot and compare number1 with number2!

number1 = 3
number2 = 5
print(number1 < > number2)
True  False

See that? We can use > and < to check if number1 is greater or less than number2.

Psst: because the results of these comparisons are either True or False, they're also known as boolean expressions.
