We've already put lists in lists and loops in loops. This time, let's put functions in functions!

def say_hi(name):
  print("Hey there, " + name + "!")

def greet(name, adj):
  say_hi(name)
  print("You look " + adj + "!")

greet( "Luke", "great today" "Leia" )
Hey there, Luke!
You look great today!

Great! Every time we call greet(), we also call the say_hi() function, a practice that we also call nesting.
