Let's create a film dictionary, shall we?

film1 = ["Brazil", "Gilliam", 7.5]
film2 = ["Jaws", "Spielberg", 8]
film3 = ["Pi", "Aranovsky", 7.5]
film4 = ["Monty Python and the Holy Grail", "Gilliam", 8]

films = [film1, film2, film3, film4]
film_dict = {}
for i in range(len(films)): 
  film_dict[films[i][0]] = films[i][1:3]
print(film_dict["Pi"])
['Aranovsky', 7.5]

In this example, we used lists about each film and a for loop to rearrange the data into a dictionary we called film_dict.
