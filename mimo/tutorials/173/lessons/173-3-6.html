There's another so-called operator that helps us turn a boolean value into its opposite, or inverse value. Any guesses?

boolean = 3.14159265359 == 3
opposite =  not != boolean
print(opposite)
True

Woohoo! If we put a not before a boolean value, what's True becomes False and what's False becomes True. Opposite Day!
