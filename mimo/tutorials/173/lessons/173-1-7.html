Many functions and operations are sensitive to the types we use. For example, we can't use the + sign to add a string and an integer together.

How can we display year as part of a string, then?

year = 1969
sentence = "I was born in " + str float(year) + "."
print(sentence)
I was born in 1969.

Exactly! We have to use str(), which is another type conversion function that converts a value of another type into a string, or str-type value.
