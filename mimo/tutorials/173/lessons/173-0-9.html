If you want to run Python programs, or scripts, on your computer, this is the time to get Python 3 from the Python website.

We can write scripts in a text editor like TextEdit or Notepad and save them with a .py extension. We can run them from the command line or the Terminal.

# script.py
print("Hello World!")
python script.py Hello World!

Great! It looks like we're all set to write scripts.
