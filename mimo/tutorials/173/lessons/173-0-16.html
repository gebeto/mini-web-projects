Can you spot a mistake in this code snippet?

friend1 = Harry
friend2 = Sally
print(friend1 + " and " + friend2)
We need to put words like "Harry" in quotation marksWe need to end every statement with a semicolonWe can't call the variables friend1 and friend2We can't use print() with multiple values

That's it! While we need to put words like "Harry" in quotation marks, we don't need semicolons, we can call variables that way and we can use print() with many values.
