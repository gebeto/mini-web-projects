We can put multiple conditions after a single if keyword. The and and or keywords help us connect them.

boring = False
beloved =  True False
if not boring or beloved:
  print("watch movie")
if not boring and beloved:
  print("but ask them first")

See that? For the so-called logical and, both conditions have to be True. For the logical or, just one of them has to be true.
