If we define the variables in our structure with var, we can change them at any time; if we use let, they're permanent.

struct Resolution {
  let width, height : Int  = 1
}
var hd = Resolution(width: 1280,
  height: 720)
print("\(hd.width)x\(hd.height)")
1280x720

Fantastic! As width and height are constants, it'd be a bad idea to set initial values for the properties.

As width and height are constants and hold the value 0, we can no longer change them with the default initializer.
