Can you anticipiate the output of this code snippet?

struct Resolution {
  var width = 0
  var height: Int {
    return Int(Double(width)
      * 0.5625)
  }
}
var r = Resolution(width: 1280)
print("\(r.width)x\(r.height)")
1280x7200x01920x10802560x1440

Good stuff! We can define computed properties that process information before they return something.

Psst: Double(width) converts width into a double value, which is then multiplied by 0.5625 and converted back in to an integer value.
