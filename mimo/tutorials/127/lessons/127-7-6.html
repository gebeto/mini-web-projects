A while loop is executed as long as a certain condition is met.

var myCounter = 0
while myCounter < 2 3 4 {
  print(myCounter, terminator:" ")
  myCounter += 1
}
print("Done!")
0 1 Done! 0 1 2 Done! 0 1 2 3 Done!

See? The code block runs while the condition is true. When myCounter reaches the threshold, the condition becomes false and the loop stops.

Careful, though: we need to be sure that the condition becomes false at some point or the loop becomes infinite.
