Whenever we use numbers as raw values, Swift does the counting. If we use Character or String types, we need to set the raw values.

enum Grade: String Character {
  case a = "A", b = "B", c = "C"
  case d = "D", e = "E", f = "F"
}
var grade = Grade.e
print(grade.rawValue) E E

Yup, we need to set a raw value for every case.
