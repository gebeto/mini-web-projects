Variables can store a variety of things. We've seen that we can store names, which we call strings, but we can also store numbers.

var myString = "4"
var myNumber = "4" 4

Great! This way, myString holds a string value and myNumber the value 4. Do you notice the difference?

Psst: if we want a variable to hold a meaningful value, we use a number without quotation marks.
