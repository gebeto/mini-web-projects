func totalize(n: Int...) -> Int {
  var total = 0
  for number in n {
    total += number
  }
  return total
}totalize(n: 8)totalize(n: 8, 1, 5)totalize(8)totalize(8.0, 1.0, 5.0)

Perfect! Functions always treat a variable number of arguments (...) as an array. Also, if we don't add an underscore, we have to use the parameter name.
