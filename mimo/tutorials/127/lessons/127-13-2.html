Alright, we have a variable that holds our function. Now what? We'll call the function with the funcVar variable.

func square(number: Int) -> Int {
  return number * number
}
let funcVar = square
print(funcVar(number:  1 2 3 )) 1 4 9

See that? In order to call a function through a variable that holds it, we use the variable name with parentheses, parameter names and arguments.
