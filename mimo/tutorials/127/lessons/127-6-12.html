A string can tell us whether it starts with a particular string value, a so-called prefix.

let str = "Mr. Boole" "Mrs. Lovelace" String()
if str.hasPrefix("Mr.") {
  print("Hello sir!")
} else if str.hasPrefix("Mrs.") {
  print("Hello ma'am!")
} else {
  print("Hello!")
} Hello sir! Hello ma'am! Hello!
