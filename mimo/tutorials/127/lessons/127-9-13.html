A class bundles properties and behavior that objects of a kind have in common.

class Spaceship {
  var fuel = 100, mileage = 0
  func travel(lightyears ly: Int) {
    fuel -= 2 * ly
    mileage += ly
    print("fuel: \(fuel)")
    print("mileage: \(mileage)")
  }
}
var myShip = Spaceship()
myShip.travel(lightyears: 5) fuel: 90
mileage: 5

Classes are like blueprints that help us make code reusable and understandable.
