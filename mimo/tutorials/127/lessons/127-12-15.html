Sometimes, raw values just aren't enough. What if we want to assign a value when we create an instance of an enumeration?

enum Product {
  case hardware(String)
  case software(String)
}
var myProduct: Product
myProduct = .hardware( "Apple" 10 )

Woohoo! This way, we can define an associated value at the moment we create variable.
