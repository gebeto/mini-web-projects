What if we don't need the whole array but a particular value in it?

let myLetters = ["a", "b", "c"]
let myLetter = myLetters [0] [1] [2]
print(myLetter) a b c

See that? We can find and retrieve values with their index.

Remember, though, that arrays are zero-based: the first value has an index of 0, the second an index of 1 and so on.
