Objects don't just have properties, they also have behavior.

class Car {
  var engineStarted = false
  func startEngine() {
    print("Vrooom!")
    engineStarted = true
  }
}
var myCar = Car()
myCar. engineStarted startEngine() 

Sweet! While variables represent an object's properties, functions take care of its behavior.
