For some strange reason, some experienced heroes tend to have their signature battle cry.

Add a function to the Hero class that returns the instance's battle cry if the level is greater than 5 and, otherwise, returns nil.

class Hero {
  // Code from earlier, plus:
  let signatureCry: String?
  func battleCry() -> String? {
    if self.level > 5 {
      return self.signatureCry
    } else {
      return nil
    }
  }
}

See that? If the level is greater than 5, battleCry() returns the signature battle cry, which may also be nil; otherwise, it returns nil.
