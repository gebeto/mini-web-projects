A function within a function? Some would call that Inception but it's possible.

func walk(steps: Int) {
  var walked = 0
  func addSteps(steps: Int) {
    walked += steps
  }
  addSteps(steps:steps)
  print("Walked \(walked) steps")
}
walk(steps:  3 4 5 ) Walked 3 steps Walked 4 steps Walked 5 steps

Did you notice anything odd? Yup, nested functions like addSteps have access to variables of their surrounding functions.
