"Sir Lancelot" has characteristics that he shares with other heroes: a name and a level as well as damage and health points.

Create a class for heroes like "Lancelot"; add an initializer that takes a name and sets property values.

class Hero {
  var name: String
  var level, damage, health: Int
  init(name: String) {
    self.name = name
    self.level = 0
    self.damage = 2
    self.health = 10
  }
}
var h = Hero(name: "Sir Lancelot")
