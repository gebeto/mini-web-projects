For the sake of completeness, we can also pass and return dictionaries.

func createDict() ->  [Int: String] [String: Int] {
  return [1: "a", 2: "b"]
}
let myDict = createDict()
print(myDict) [2: "b", 1: "a"]

Great! As dictionaries are unordered, we don't need to worry about the order of the key-value pairs.
