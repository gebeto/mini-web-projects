Let's use a for loop to sum up a bunch of numbers!

var myNumber = 0
print(myNumber, terminator: "")
for i in 1 ... 2 3 4 {
  myNumber += i
  print("+\(i)", terminator: "")
}
print("=\(myNumber)") 0+1+2=3 0+1+2+3=6 0+1+2+3+4=10

Nice! i, the index variable, is added to myNumber in every iteration of the loop.
