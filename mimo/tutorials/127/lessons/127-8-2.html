If we need a code block more often, it's a good idea to make it a function. Once we have it, we can always reuse it.

func sayHi(to: String) {
  print("Hi \(to)!")
}
let person1 = "Phil"
let person2 = ""
sayHi(to: person1)
sayHi(to: person2) Hi Phil!
Hi !

When we define a function, we can optionally define parameters, which are values that the function treats as input from outside.
