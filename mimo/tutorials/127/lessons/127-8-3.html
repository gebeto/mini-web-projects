Parameters help us adapt the behavior of functions to different situations.

func sayHi(to: String, and: String) {
  let str = "Hi \(to) and \(and)!"
  print(str)
}
let person1 = "Ben"
let person2 = "Leia"
sayHi(to: person1, and: person2) Hi Ben and Leia!

If we want to use multiple parameters, we need to separate them with commas.
