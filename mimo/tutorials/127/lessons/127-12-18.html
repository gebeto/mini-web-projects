Enumerations are a great way to represent a fixed set of constants.

enum Weekend {
  case saturday, sunday
  var description: String {
    switch self {
    case .saturday:
      return "go-out day"
    case .sunday:
      return "funday"
    }
  }
}
var day = Weekend.sunday
print(day.description) funday

A Weekend-type variable can only hold the values Weekend.saturday and Weekend.sunday.
