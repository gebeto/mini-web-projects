If we take care of every possible case, we don't need to define a default case.

enum Season {
  case spring, summer, fall, winter
  func description() -> String {
    switch self {
    case .summer:
      return "hot season"
    case .winter:
      return "cold season"
    case .spring, .fall:
      return "transitional season"
    }
  }
}
var season = Season. spring fall
print(season.description()) transitional season transitional season

See that? Because spring and fall are grouped together, they have the same description.
