We just created a variable, called it name and gave it a value.

Variables allow us to store values for later. We create them with the word var, followed by a name, an = sign and a value.

var task = "Learn how to code"
task = "Scout for worms" "Sit on a treebranch"
print(task) Scout for worms Sit on a treebranch

Just like I can't do more than one of these things at once, a variable can only remember a single value at a time.

If we want to change a variable's value, we simply skip the var part.
