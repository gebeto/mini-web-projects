In some cases, a variable number of parameters comes in handy.

func totalize(numbers: Int...) {
  var total = 0
  for n in numbers {
    total += n
  }
  print(total)
}
totalize(numbers:  1, 4, 9 1, 4, 9, 27 1, 4, 9, 27, 38) 14 41 79

See? Within the function, the parameters are treated like values of an array.
