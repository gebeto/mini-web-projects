"Sir Lancelot" can use his bow, fists, spear or sword to resolve disagreements.

Create an enumeration with the possible weapons and don't forget to provide a function or computed property to describe the weapons.

enum Weapon {
  case bow, fist, spear, sword
  var description: String {
    switch self {
    case .bow:
      return "long distance"
    // The other cases, plus:
    default:
      return "nothing left"
    }
  }
}

With enumerations, we get more control over the values that variables can take.
