Normally, we need to use the name of every parameter when we call a function. If that's not what we want, we add an underscore before it and skip the name.

func totalize(_ numbers: Int...) {
  var total = 0
  for n in numbers {
    total += n
  }
  print(total)
}
totalize( 8, 1, 5 numbers: 8, 1, 5) 14

That's right! If the function's name describes a parameter well enough (as in print()), we can add an underscore and skip the name.

Here's the thing: if we add an underscore to the parameter, we have to skip it when we call the function.
