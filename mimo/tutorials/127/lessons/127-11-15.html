We can check whether a function was executed or not.

class Human {
  func think() {
    // Black-box processes ...
  }
}
var me:  Human? Human? = Human()
if me?.think() != nil {
  print("Executed")
} else {
  print("Not executed")
}
Not executed Executed

See that? If me is has a value, think() is executed; otherwise, it's not executed.
