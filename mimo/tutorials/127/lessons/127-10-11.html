This time, "Sir Lancelot" is breaking his way through the wilderness. But where exactly is he?

Create a structure for the latitude, longitude and altitude in his world; add a property of the sort to the Hero class and give it a value.

struct Position {
  var x, y, z: Double
}class Hero {
  // ...
  var position = Position(x: 0.0,
    y: 0.0, z: 0.0)
}

With a position property, "Sir Lancelot" can keep track of his location in the wilderness.
