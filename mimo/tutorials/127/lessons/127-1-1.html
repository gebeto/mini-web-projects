If the value of a variable shouldn't or won't change, it's a good idea to make it a constant.

var let myGoal = 10
var current = 0

Sweet! This way, we can only change the value of current because myGoal is a constant.

It's good practice to use let by default and go for var in case we might need to change a variable's value.
