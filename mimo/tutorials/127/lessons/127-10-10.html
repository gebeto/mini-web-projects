In general, it's a really good idea to provide comments with meaningful explanations.

struct Position {
  var x = 0.0, y = 0.0
  //  Return string value of coordinates We're so good at this!
  var description: String {
    return "[\(x)/, \(y)]"
  }
}
var position = Position()
print(position.description)
[0.0, 0.0]

Good stuff! Comments help others (and ourselves) understand the purpose of complex expressions and processes.
