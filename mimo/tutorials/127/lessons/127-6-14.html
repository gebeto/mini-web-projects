"Sir Lancelot" is very confused (see: fugue state). He has leveled up so quickly that he needs some reassuring feedback!

Create name, occupation and level variables (or constants, if applicable). Then, use string interpolation to create a single-string summary of them.

let name = "Sir Lancelot"
let occupation = "Knight"
var level = 6
var str = "\(name),\(occupation); "
str += "current level: \(level)"
print(str) Sir Lancelot, Knight; current level: 6

Great! This way, a confused "Sir Lancelot" can always check his name and level.
