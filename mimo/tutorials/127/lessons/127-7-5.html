We can also use this kind of loop to iterate over the keys-value pairs of a dictionary.

let myDict = [1 : "a", 2 : "b"]
for (key, value) in myDict {
  print(key, terminator: "->")
  print(value, terminator:" ")
} 1->a 2->b

Iterating over a dictionary isn't any harder than going through the values of an array, is it?
