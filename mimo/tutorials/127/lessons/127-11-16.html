If, however, we return an optional and it's nil, the code in the if block can't be executed.

class Human {
  func think() -> Int? {
    return 42 nil
  }
}
var me: Human? = Human()
if me?.think() != nil {
  print("Executed")
} else {
  print("Not executed")
}
Executed Not executed

In this case, the if block is only executed if we return a value that isn't nil.
