We can also use optional chaining to call a function, provided the optional holds a value.

class Human {
  func say(something: String) {
    print(something)
  }
}
var me: Human? = Human()
me ? !.say(something:
  "I'm giving up on you") I'm giving up on you

Sweet! Because me has a value, say() is called.

And, of course, I'm not giving up on you; you're doing great!
