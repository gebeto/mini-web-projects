We can require initial values for the structure as well.

struct Resolution {
  var width, height: Int
  init(width: Int, height: Int) {
    self.width = width
    self.height = height
  }
}
let vga = Resolution(width: 640, 
  height: 480)
print(vga)
Resolution(width: 640, height: 480)

We can assign initial values with an initializer. Again, we've seen this in classes.
