So, why is this kind of value that can only be 1 or 0 called a boolean value (or a Bool type)?

let george = true
let boole = false

It's named after George Boole, an English mathematician and philosopher. Boole invented a branch of algebra, in which a value can only be 1 or 0, true or false.
