When elements are floated, they go from stacking on top of each other to the left or right until they reach the boundaries of their parent or another floated element.

.box {
 width: 200px;
 height: 200px;
 float:  left none;
}
.blue { background: #44accf; }
.green { background: #b7d84b; }
.orange { background: #e2a741; }
.pink { background: #ee3e64; }

Sweet! The boxes line up in a row and, depending on the size of the browser window, they continue in a new row if there's not enough room.
