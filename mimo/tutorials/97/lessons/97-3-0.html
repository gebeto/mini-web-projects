In this chapter, we'll talk some more about design and layout.

The so-called box model means that there's a box that wraps around every HTML element and describes the amount of space the element takes up.

p div {
 background-color: #44b3c2;
 width: 150px;
 padding: 25px;
 border: 25px solid #f1a94e;
 margin: 25px;
}
<!doctype html>
<html>
 <head>
  <link rel="stylesheet" type="text/css" href="styles.css">
 </head>
 <body>
  <p>I'm the actual content.</p>
 </body>
</html>
<html>
 <head><style>p {background-color: #44b3c2;width: 150px;padding: 25px;border: 25px solid #f1a94e;margin: 25px;}
  </style>
 </head>

</html>
]]>
