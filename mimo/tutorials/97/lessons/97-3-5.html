Margins are the outermost part of the box model, creating white space around an element and outside of its borders that separates it from its neighbors.

p {
 border: 2px solid #44b3c2;
 margin-top: 10px;
  margin-right padding-right: 25px;
 margin-bottom: 10px;
 margin-left: 25px;
}
<!doctype html>
<html>
 <head>
  <link rel="stylesheet" type="text/css" href="styles.css">
 </head>
 <body>
  <p>I'm the actual content.</p>
 </body>
</html>
<html>
 <head><style>p {border: 2px solid #44b3c2;margin-top: 10px;margin-right: 25px;margin-bottom: 10px;margin-left: 25px;}
</style></head><body><p>I'm the actual content.</p></body></html>
]]>

Fantastic! We can set the white space outside of an element's borders for every side of the element.
