Often times, however, we want a style to apply to whole web sites.

<!doctype html>
<html>
 <head>
  <title>Web page 1</title>
  <style>
   h1 { font-family: Verdana; }
   p { font-size: 12px; }
  </style>
 </head>
</html>
<!doctype html>
<html>
 <head>
  <title>Web page 2</title>
  <style>
   h1 { font-family: Verdana; }
   p {  font-size font-family: 12px; }
  </style>
</head>
</html>

If, for example, we wanted to change the font family for every heading on the web site to Verdana, we'd have to go through every single web page.

Psst: web sites are collections of web pages.
