Let's use the properties and units we've discussed to improve your web page!

Set the body section's font family to "Lucida Grande", define a fallback and make the font twice the default size. Define pixel heights for the header and the footer.

body {
 font: 2em "Lucida Grande", sans-serif;
}
#header {
 background-color: #44b3c2;
 height: 150px;
}
#footer {
 background-color: #f1a94e;
 height: 50px;
}
<html>
 <head>
  <title>Henry's web page</title>
<style>body {font: 2em "Lucida Grande",sans-serif;}#header {background-color: #44b3c2;height: 150px;}#footer {background-color: #f1a94e;height: 50px;}</style></head>
 <body><div id="header"><h1>Henry's web site</h1></div><div id="content"><p>This is the home page of my web site.</p></div><div id="footer"></div>
</body>
</html>
]]>

Huzzah! We've even updated the colors to somewhat more beautiful hexadecimal colors.
