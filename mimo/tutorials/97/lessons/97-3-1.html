By default, block-level elements take up the full width of their surrounding elements. Using the different display value, however, we can change that.

.box {
 background-color: gray;
 width: 50px;
 height: 25px;
 display:  inline block;
}
<!doctype html>
<html>
 <head>
  <link rel="stylesheet" type="text/css" href="styles.css">
 </head>
 <body>
  <div class="box"></div>
  <div class="box"></div>
 </body>
</html>
<html>
 <head><style>.box {background-color: gray;width: 50px;height:25px;display: inline;}</style>
 </head>
 <body><div class="box"></div><div class="box"></div></body>
</html>
]]>
