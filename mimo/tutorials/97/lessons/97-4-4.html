Aside from wrapping text around images, the float property is incredibly useful to create layouts.

#header {
 background: #44accf;
 height: 50px;
}
#content {
 background: #b7d84b;
 width: 75%;
 height: 500px;
 float: left;
}
#navigation {
 background: #e2a741;
 width: 25%;
 height: 500px;
 float: right;
}
#footer {
 background: #ee3e64;
 height: 50px;
 clear: both left right;
}
<!doctype html>
<html>

 </head>
 <body>
  <div id="header"></div>
  <div id="content"></div>
  <div id="navigation"></div>

 </body>
</html>

Gorgeous! By setting the clear property to both, we're telling the footer that it needs to sit below any floated element.
