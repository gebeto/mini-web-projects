That said, how can we make the orange box show up again? Well, we can use clear to put it back into the normal flow. Does that sound like plan?

.box {
 width: 200px;
 height: 200px;
}
.blue { background: #44accf; }
.green { background: #b7d84b; }
.orange { background: #e2a741; }
.pink { background: #ee3e64; }
.floated { float: left; }
.cleared { clear float: left; }
