Let's get back to that web page of yours and make it a little more web page-y!

Add an unordered list of three links to the header section and remove the list's padding. Display the list items as inline-blocks and give them a 10px padding.

/* ... */
ul { padding: 0px; }
li {
 display: inline-block; 
 padding: 10px;
}
<!doctype html>
<html>
 <!-- ... -->
 <body>
  <div id="header">
   <h1 id="title">Henry's web page</h1>
   <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Contact</a></li>
   </ul>
  </div>
  <!-- ... -->
</html>
<html>
 <head>
  <title>Henry's web page</title>
  <style>body {font: 2em "Lucida Grande", sans-serif;}#header {background-color: #44b3c2;height: 150px;}#footer {background-color: #f1a94e;height: 50px;}ul { padding: 0px; }li {display: inline-block; padding: 10px;}</style></head>
 <body><div id="header"><h1 id="title">Henry's web page</h1<ul><li><a href="#">Home</a></li><li><a href="#">Blog</a></li><li><a href="#">Contact</a></li></ul></div><div id="content"><p>This is the home page of my web site.</p></div><div id="footer"></div></body></html>
]]>

Wow, we just turned an unordered list into a navigation bar! Let's not forget to provide some meaningful values for the href attributes at a later stage!
