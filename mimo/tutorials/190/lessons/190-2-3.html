Whenever we commit something, it'll be committed to the currently checked-out branch.

$ cat cookie.txt
Chocolate Chip Cookie
$ git branch alternate_reality
$ git checkout alternate_reality master 
$ echo Frosted Snowman Cookie > cookie.txt
$ git add .
$ git commit -m "Modify cookie"
[alternate_reality 7d6808a] Modify cookie
 1 file changed, 1 insertion(+), 1 deletion(-) [master a8ed121] Modify cookie
 1 file changed, 1 insertion(+), 1 deletion(-)

See that? If we switch to alternate_reality, the commit will go to that branch. If we stay on master, so will the commit.
