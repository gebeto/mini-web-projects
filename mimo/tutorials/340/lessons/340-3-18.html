But wait! Copy-pasting identical code isn't very smart. Let's get rid of the original code in viewDidLoad() and call the newRandomNumber() function instead.

override func viewDidLoad() {
super.viewDidLoad()
self.newRandomNumber(self)
}

We're passing self as an argument because the view controller is the sender. When we trigger the function by tapping on the button, the button will act as the sender.
