Within a database, a table is a collection of data that's organized into columns and rows. Does that sound familiar? Might friends be a table?

SELECT *
FROM friends name;  id | name  | age  | height 
----+-------+------+--------

Indeed, friends is a table and id, name, age and height  are its columns. A row in this table represents a unique friend and is referred to as a record.
