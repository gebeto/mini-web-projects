Negation operators look for results that don't match a condition.

SELECT *
FROM friends 
WHERE age  != ! 99;
 id | name  
----+-------
  1 | Robin 

Great! For comparisons, we can combine ! with other operators. != is not equal to, just as !&gt; is not greater than and !&lt; is not less than.

Psst: for logical expressions, we can use the NOT operator.
