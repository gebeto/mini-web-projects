Tables are a fundamental concept of SQL. We've learned how to create, alter and delete them and have seen ways to work with their records.

CREATE TABLE team (
 number INT NOT NULL,
 name VARCHAR(20)
);
INSERT INTO team
VALUES (23, 'Michael');
SELECT number, name
FROM team
WHERE number = 23;
 number | name    
--------+---------
     23 | Michael
