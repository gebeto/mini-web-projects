So, in reality, statements are made up of clauses. In a select statement, we use the SELECT clause for the columns and the FROM clause for the tables.

What about this clause, though?

SELECT name
FROM friends
WHERE name =  'Robin' Robin;
 name  
-------
 Robin 

Nice! We can use a WHERE clause to get results based on certain conditions. If we skip it, we get every row of a table.

Psst: some database systems require single quotes for values, which is why we'll prefer them over double quotes.
