Remember our library? Can you write a statement that'll return all the records that we haven't read yet?

SELECT *
FROM books
WHERE read = FALSE;
 id |        title         | read

  9 | The Color Purple     | f

Whoa, look at how much we've got left to read! In this example, SQL saved the BOOL-type values of read as t and f.
