We can also use = as a comparison operator, though.

SELECT *
FROM friends 
WHERE age  = EQUALS 24;
 id | name  | age | height
----+-------+-----+--------
  1 | Robin |  24 |   1.72 

Whoop! In this context, the = operator compares if age is equal to 24.

Psst: because age = 24 can either be TRUE or FALSE, it's also known as a boolean statement.
