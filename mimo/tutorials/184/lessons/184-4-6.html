What do you think will be happen when we run the following code?

int ourArray[3] = {0, 100, 200};
ourArray[3] = 300;
cout << ourArray[3];
300

ourArray only has space for 3 elements, but if we insert a value into the 4th spot, it runs fine and we can even output the element!

C++ doesn't check where a new element fits in an array and we could lose data if is put into memory already occupied!
