It's also possible to have an if block within another if block. Take a look.

int x = 10;
int y =  12 9 ;
if(x == 10)
{
 if(y > 10)
  {
   cout << "We're in the nest!";
   cout << endl;
  }
 cout << "We're outside the nest!";
}
We're in the nest!
We're outside the nest! We're outside the nest!

Nice. Make sure you try both options before you move on.

If we enter the first block and the second condition is true, both messages will output. Otherwise we'll only get the one.
