What if we want to change a variable based on its current value?

Yes, we can work with them and assign them back to themselves but we can also use a shortcut.

int speed = 20;
speed = speed + 10;
speed = speed - 10;
speed  += -= *= /= 10;
cout << speed;
30 10 200 2

Nice! These compound assignment operators, use the value that's stored in the variable before them and assign it back to the variable.

Psst: the += operator also works for strings.
