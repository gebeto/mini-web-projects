Now, what should we do with these numbers? Let's try some arithmetic operations!

int number1 = 9;
double number2 = 9.12345;
cout << number1  + - * / number2;
18.1234 -0.12345 82.1111 0.986469

Wow! All the usefulness of arithmetic with none of the pain. C++ can be pretty darn handy!

Psst: while we can't use the -, * and / signs with strings, we can use the + operator to add them together.
