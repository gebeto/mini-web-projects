Just one more: the logical or operator || tells us if at least one of our expressions is true. It doesn't care about the rest.

bool b1(12  > < 15);
bool b2(21 < 12);
bool b3(10 == 13);
bool bResult(b1 || b2 || b3);
cout << boolalpha << bResult;
false true

Awesome. If only one expression is true, the return value is true. See what happens if all expressions are false?
