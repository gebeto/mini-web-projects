A loop is another form of the control flow we saw earlier. If we want to run a block of code repeatedly while a certain condition is true.

int arr[4] = {0, 1, 2, 3};
int index = 0;
cout << arr[index];
index++;
cout << arr[index];
index++;
cout << arr[index];
index++;
cout << arr[index];
