In this chapter, we've seen how we declare and initialize variables, how to reassign them as well as ways to change values.

auto distance(30);
distance += 10;
int time, speed;
time = 45;
speed = 65;
cout << "It will take " << time << " to travel " << distance << " miles, at " << speed << " mph.";

Great! Variables are a great help, as long as we don't forget to initialize them.
