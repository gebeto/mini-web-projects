Even if we make it impossible for users to type innumbers, they can still copy-paste them into the text field. Let's make sure that the input really is a number!

As part of the optional binding, try to transform the input variable into an Int-type variable.

@IBAction func guess(_ sender: Any) {
if let input = self.inputField.text, let number = Int(input) {
print("Input: \(number)")
} else {
print("No input")
}
}
