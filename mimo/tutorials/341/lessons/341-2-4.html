In order to analyze the text in the text field, we can ask for the field's text value when the user taps on the Guess button.

Create a local input variable in the guess() function and assign the text value of the inputField instance variable to it.

import UIKit

class GameViewController: UIViewController {

@IBOutlet weak var inputField: UITextField!

@IBAction func guess(_ sender: Any) {
let input = self.inputField.text
}

}
