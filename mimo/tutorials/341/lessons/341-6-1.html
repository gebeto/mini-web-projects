Let's have the guess() function return the GuessAnswerenumeration instead of a boolean!

Define the return type as GuessAnswer and use an if statement to check if the guess was correct, too low, or too high.

func guess(_ guess: Int) -&gt; GuessAnswer {
self.tries += 1
if number == guess {
return .correct
} else if number &gt; guess {
return .tooLow
} else {
return .tooHigh
}
}
