func showAlert() {
  let alertController = UIAlertController(title: "Error", message: "Please enter a number!", preferredStyle: .alert)

  let defaultAction = UIAlertAction(title: "OK", style: .default, handler: nil)
  alertController.addAction(defaultAction)

  present(alertController, animated: true, completion: nil)
}

We initialize alertController and pass title, message, and preferredStyle arguments to the initializer. Then we initialize defaultAction and pass some action-related arguments to its initializer. We then add the action to the controller and, finally, we present the controller.
