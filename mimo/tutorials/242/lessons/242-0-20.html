Isn't it time to work on actual ArnoldC projects?

First up, get ArnoldC.jar from its GitHub page. Open a text editor, create a text file that you populate with ArnoldC code and save as project.arnoldc. Open a Terminal window and enter:

$ java -jar ArnoldC.jar project.arnoldc
$ java project

Done! We compile project.arnoldc using ArnoldC.jar and run the project program using the java command.
