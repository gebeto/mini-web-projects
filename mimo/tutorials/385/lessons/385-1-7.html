How do we transform the links to uppercase letters, though? We don't want to change the actual text within the &lt;a&gt; elements, do we?

In the #header a rule, set text-transform to uppercase and, to give the letters some more space, letter-spacing to 0.1em.

...
#header a {
  color: white;
  text-decoration: none;

  text-transform: uppercase;
  letter-spacing: 0.1em;
}
...
