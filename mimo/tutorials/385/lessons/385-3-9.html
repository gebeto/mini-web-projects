Isn't it about time that we use a nicer font for the webpage? For this blog, we'll use a web font called Montserrat.

At the top of styles.css, add an @import rule to, well, import the web font.

@import url(http://fonts.googleapis.com/css?family=Montserrat:400,700);

Psst: 400 and 700 stand for normal and bold typefaces.
