Sometimes we want to get rid of values. Let's try to remove something!

HashMap<Integer, String> map;
map = new HashMap<>();
map.put(23, "Michael");
map.put(91, "Dennis");

map.remove( 23 91 );
System.out.print(map); {91=Dennis} {23=Michael}

See that? In order to get rid of values with remove(), we need to have a key that we can pass to the method.
