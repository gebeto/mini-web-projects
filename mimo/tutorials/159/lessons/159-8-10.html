Loops repeat blocks of code for as often as we want. That's very convenient when it comes to repetitive tasks.

String[] friends = { "Harry",
 "Ron", "Hermione" };
for (String f : friends) {
 System.out.print(f + "; ");
}
int year = 1;
while (year <= 7) {
 System.out.print(year);
 year++;
}

Pure magic! But let's not get caught in an infinite loop and move on!
