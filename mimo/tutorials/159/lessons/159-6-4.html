Retrieving, or getting, an element of an ArrayList is very straight-forward. Why? Because there's a method that does just that.

ArrayList<String> names;
names = new ArrayList<>();
names.add("Jack");
names.add("Kate");
String name = names.get( 0 1 2 );
System.out.print(name); Jack Kate

See that? Just like arrays, ArrayLists are zero-based: the first element has an index of 0, the second has an index of 1 and so on.

Are you sure that there's an element with a zero-based index of 2?
