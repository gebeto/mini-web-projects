An ArrayList is an ordered list that holds a variable number of values of objects or specified types of objects.

ArrayList<String> courses;
courses = new ArrayList<>();
courses.add("Javvy");
courses.add("Swift");
courses.set(0, "Java");

String s = String.join(", ", courses);
System.out.print(s); Java, Swift

ArrayLists are a great alternative to arrays if we can't say how many values we'll have to store down the line.
