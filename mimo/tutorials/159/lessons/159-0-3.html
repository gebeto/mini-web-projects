Variables can remember words, which we call strings, but also numbers.

String myString = "4";
int myNumber = "4" 4 ;

Sweet! Now myString contains the string "4" and myNumber the number 4.

We create string variables with the word String and put their values in quotation marks. Numbers use different words and no quotation marks.

Did you notice the quotation marks that we also used with myString? Might that be how we create string variables and not numbers?
