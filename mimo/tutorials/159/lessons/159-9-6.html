Sometimes, however, we don't know exactly how many values, or arguments, we need to pass to a method.

int totalize(int... numbers) {
 int result = 0;
 for (int n : numbers) {
  result += n;
 }
 return result;
}

int s = totalize( 8, 1, 5 4, 8, 15, 16, 23, 42);
System.out.print(s); 14 108

That's where ... comes in handy because it allows us to pass a variable number of arguments. These arguments are treated as elements of an array.
