We can pull off some great tricks with strings. For example, we can check if a string starts with a specific String-type value.

String idol = "Mr. Gosling" "Mrs. Hopper" ;
if (idol.startsWith("Mrs.")) {
 String output = "Hello ma'am!";
 System.out.print(output);
} else if (idol.startsWith("Mr.")) {
 String output = "Hello sir!";
 System.out.print(output);
} Hello sir! Hello ma'am!

See that? startsWith() returns true or false, based on whether or not the specified String-type value is a so-called prefix of the string.

Psst: James Gosling is the original creator of Java and Grace Hopper developed the first compiler for a computer programming language.
