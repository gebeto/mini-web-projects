Sometimes it makes sense to return the result of a method.

int triple(int number) {
 int result = number * 3;
 return result;
}

int myNumber = triple( 13 27 59 97 );
System.out.print(myNumber); 39 81 177 291

See that? Whenever we want to return something, for example a result of a calculation, we need to use the return value's type instead of void.
