Methods make code more readable and reusable. They're handy and easy to use.

String reverse(String a) {
 String result = "";
 int l = a.length();
 for (int i = 0; i < l; i++) {
  result = a.charAt(i) + result;
 }
 return result;
}

String str = reverse("REDRUM");
