Can you get this code to display "1-22-333-"?

for (int i=1; i<=3; i++) { for (int j=1; j<=i; j++) {  System.out.print(i); } System.out.print("-");}

Great! We open the outer loop to count from 1 to 3. Then we open the inner loop to run for i times and print the numbers. After closing the inner loop, we print "-" and close the outer loop.
