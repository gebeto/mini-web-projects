Villains generally have a name and weakness that "Agent Smith" can use to defeat their evil plans.

Create a Villain class with a custom constructor that takes and stores a name and a weakness. Also, provide a method that returns a description of the villain.

class Villain
{
 private String name;
 private String weakness;
 
 public Villain(String name, String weakness)
 {
  this.name = name;
  this.weakness = weakness;
 }
 public string GetDetails()
 {
  return "The villain's name is " + name + " and his/her 
weakness is " + weakness;
 }
}

This class will come in handy when "Agent Smith" faces a villain in the future.
