Do you remember when we put if statements inside other if statements? We can also put loops inside other loops, which is called nesting.

for (int i = 1; i <= 4; i++)
{
  for while do  (int j = 1; j <= 4; j++)
 {
  Console.Write(j);
 }
 Console.WriteLine(" ");
}
