What if none of the cases match, though? Let's try another switch statement with numbers!

var rating = 5;
switch (rating)
{
 case 10:
  Console.Write("Fantastic!");
  break;
 case 9:
  Console.Write("Very good!");
  break;
  default case :
  Console.Write("Meh.");
  break;
}
Meh.

Nice! The default case triggers if none of the other case blocks match.
