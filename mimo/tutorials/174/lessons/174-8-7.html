The break keyword is a so-called control statement. Let's look at another such statement: continue!

int i = 0;
while (i < 10)
{
 i++;
 if (i > 1 && i < 5)
 {
  continue break ;
 }
 Console.Write(i + "; ");
}
1; 5; 6; 7; 8; 9; 10; 

Fantastic! When i is greater than 1 and less than 5, the continue keyword skips the rest of the so-called iteration but doesn't stop the loop itself.
