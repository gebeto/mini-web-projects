When the if statement returns false, we can skip ahead to another block of code.

if ( 5 > 6 5 == 6 5 >= 6)
{
 Console.WriteLine("Hey!");
}
else 
{
 Console.WriteLine("Ho!");
}
Ho! Ho! Ho!

See that? When the boolean statement in the parentheses after the if is false, the code in the else block will run.
