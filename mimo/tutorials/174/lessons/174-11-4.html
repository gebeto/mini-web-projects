Do you remember the namespace keyword from the early days? Namespaces help us structure projects into meaningful pieces.

using System;

namespace App
{
 class Program
 {
  public static void Main(string[] args)
  {
   var c = new Classes.Car Car("red");
  }
 }
}
using System;

namespace Classes
{
 class Car
 {
  private string color;
  public Car(string color)
  {
   this.color = color;
  }
  public void StartEngine()
  {
   Console.Write("Vroom!");
  }
 }
}

Sweet! With a namespace, we can provide hints about the purpose of the classes it contains and distinguish between classes with the same name.

Might we have to provide the namespace's name as well so the Car class can be found?
