If we want an assignment to depend on a condition, we can also use the ? and : signs, which we call the conditional operator.

int hour = 10;
string greeting =  hour < 12 hour ? "Good morning" : "Hi";
Console.Write(greeting);
Good morning

Nice! If the condition is true, the value after the ? sign gets assigned; otherwise, the value after the : sign is used.
