Remember the break keyword that we used to exit switch cases? Well, guess what: we can also use it with loops.

int i = 0;
while (i < 10)
{
 i++;
 if (i >  1 3 5 )
 {
  break;
 }
 Console.Write(i + "; ");
}
1;  1; 2; 3;  1; 2; 3; 4; 5; 

See that? The loop runs while i is less than 10, but as soon as i is greater than 5, the break statement causes it to stop.
