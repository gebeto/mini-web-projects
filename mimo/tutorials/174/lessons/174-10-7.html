We've seen properties and how we can define them in a constructor.

We also touched on encapsulation. Keeping members private is a key part of that. Like we saw last lesson, if we do, we can't access them from outside the class!

class Car {
 private string color;
 
 public string Color {
  get {
   return color;
  }
  set {
   color = value;
  }
 }
}

Car vehicle = new Car();
vehicle.color = "Blue";
Console.Write(vehicle.color);
Blue

We had to use that .Describe() method just access the private color property. But now we have the get and set.

As you'd expect, get allows us to get the property with .color, and set allows us to set it with .color =.
