If we want to see all the keys in the dictionary, we can use a simple method to retrieve them as a collection.

var dict = new Dictionary<int, string>() {
 {4, "Locke"}, 
 {8, "Reyes"}
};
var s = String.Join(", ", dict. Keys Count Values);
Console.WriteLine(s);
4, 8

Sweet! Using the Keys property with String.Join(), we can convert the keys into a string-type value and write them out.
