Every C# program needs to contain a Main() method to mark the entry point of the program.

using System;
namespace SpecialMission
{
 class MyFirstApp
 {
  public static void Main(string[] args)
  {
   Console.WriteLine("");
  }
 }
}

Sweet! We'll discuss this a little later; for now, it's important to remember that the code goes between the braces of the Main() method.
