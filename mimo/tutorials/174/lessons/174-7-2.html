Accessing elements in a dictionary is a little different to arrays and lists. We access a value by using it's key.

Dictionary<string, int> dict = new Dictionary<string, int> ();
dict.Add("Ebenezer", 11);
dict.Add("Pippin", 45);
dict.Add("Estella", 37);
Console.Write(dict[" Ebenezer Pippin Estella"]);
11 45 37

See that? It's not too hard to access values.
