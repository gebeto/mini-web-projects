Which of these loops are infinite?

int i = 0;
while (i < 100)
{
 Console.Write(i);
}for (i = 0; i < 100; i++)
{
 Console.Write(i);
}int i = 0;
do {
 Console.Write(i);
} while (i > 0);

Great job! Because i will always be less than 100, the while loop is infinite. The for loop stops at 100 and the do while loop never starts.
