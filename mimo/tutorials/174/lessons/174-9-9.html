As always, play around in your development tool: create methods, use parameters and all the different things we've learned this chapter.

static string BuildString (string s1, string s2 = "there") {
 string result;
 result = s1 + " " + s2;
 return result;
}
string s = BuildString("Hi");
Console.Write(s);
