We can also put if blocks inside other if blocks. Let's display "Triggered!" in the console!

int x = 0;
if (x > -5)
{
 if ( x < 5 x == 5) 
 {
  Console.WriteLine("Triggered!");
 }
}
Triggered!

Woah, good job! We want both if blocks to be true in order to get Console.WriteLine() to run.

Does x really seem to be 5?
