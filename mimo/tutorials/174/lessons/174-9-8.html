"Agent Smith" needs to crack a safe. The number of attempts it takes him to get it open depends on the safe's security level and his equipment.

Create a CalculateAttempts() method that takes security and equipment levels, multiplies their product with 3 and returns the result.

static int CalculateAttempts(int security, int equipment)
{
 int result;
 result = security * equipment * 3;
 return result;
}
int att = CalculateAttempts(3, 3);
Console.Write(att + " attempts");
27 attempts

This method will make it a lot easier for "Agent Smith" to save the day.
