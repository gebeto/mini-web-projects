Sometimes we might want to sort the elements of an array. Lucky for us, there's a built-in method that does just that.

int[] ages = { 30, 21, 27, 24 };
Array. Sort Empty(ages);
var s = String.Join(", ", ages)
Console.WriteLine(s);
21, 24, 27, 30

Great! We can use Array.Sort() to sort the array in ascending order.
