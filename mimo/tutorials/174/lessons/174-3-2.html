When the if statement returns false, we can also run a block called else if, which will run when its condition is met.

int x = 150 50 10 ;
if (x > 100)
{
 Console.WriteLine("x > 100");
}
else if (x > 30)
{
 Console.WriteLine("x > 30");
}
else 
{
 Console.WriteLine("x <= 30");
}
x > 100 x > 30 x <= 30

Sweet! We can have many else if blocks but just a single else block.
