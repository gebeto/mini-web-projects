In a sense, classes are blueprints of objects.

class Pet {
 private string name; 
 
 public string Name {
  get {
   return name;
  }
  set {
   name = value;
  }
 }
}

Pet puppy = new Pet();
puppy.Name = "Sparky";
Console.Write(puppy.Name);
Sparky

Object orientation, which includes classes, constructors and inheritance, is a big deal in programming.
