What if we have to check for many scenarios? We can run multiple blocks of code based on different scenarios.

string grade =  "A" "B" "C" "D" ;
switch(grade)
{
 case "A":
  Console.WriteLine("Woop!");
  break;
 case "B":
  Console.WriteLine("Nice!");
  break;
 case "C":
  Console.WriteLine("Alright!");
  break;
 case "D":
  Console.WriteLine("Uh oh!");
  break;
}
Woop! Nice! Alright! Uh oh!

Depending on the value that we pass to switch, the code in the relevant case will run. The break keyword makes sure that the case has an end.
