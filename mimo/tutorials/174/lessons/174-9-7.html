Using the params keyword and an array, we can also have an unlimited number of parameters. The possibilities are endless!

static int Totalize(params int[] numbers)
{
 int result = 0;
 foreach (int number in numbers)
 {
  result += number;
 }
 return result;
}
Totalize( 1 1, 2, 3, 4, 5);
1 15

See that? From zero to an endless stream of values: there's no limit to how many values, or arguments we can pass to Totalize().
