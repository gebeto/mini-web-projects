Let's get back to a classic collection method and let go of a key-value pair in this dictionary!

var cSharp = new Dictionary<string, bool>() {
 {"Fun", true},
 {"Boring", false}
};
cSharp.Remove( "Boring" false );
var s = String.Join(", ", cSharp);
Console.WriteLine(s);
[Fun, true]

Sweet! As we've seen before, we can use the Remove() method; this time, with the key in the parentheses.
