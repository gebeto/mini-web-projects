Let's help "Agent Smith" work out how reliable some secret intelligence is. He received two dossiers, one of which has to be trustworthy in order to pass the test.

Create variables with bool-type values for the dossiers. Evaluate the dossiers and display true if they pass the test.

bool dossier1 = true;
bool dossier2 = false;
bool pass = dossier1 || dossier2;
Console.WriteLine(pass);
