So, let's see how we can remove an item from a list!

var villains = new List<string>();
villains.Add("Lex Luthor");
villains.Add("Magneto");
villains.Add("Green Goblin");
villains. Remove Add("Magneto");
var s = String.Join(", ", villains);
Console.WriteLine(s);
Lex Luthor, Green Goblin

Nice! The Remove() method gets rid of the item we specify. If we want to use the index number instead of the item's value, we can use RemoveAt().
