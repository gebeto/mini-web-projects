Booleans are a data type the values of which can only be true or false. When we compare values, we get a bool-type value in return.

bool bool1;
bool1 = true;
var bool2 = 75 >= 50;
bool bool3 = bool1 && bool2;
Console.WriteLine(bool3);
True

Does that make sense? If so, great; if not, just keep practicing and you'll get the hang of booleans and boolean operators.
