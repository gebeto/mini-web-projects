Let's play around with some permissions!

Create a project.txt file that you make writable and executable for user but only writable for group and others. Finally, list the permissions.

$ touch project.txt
$ chmod u+w project.txt
$ chmod u+x project.txt
$ chmod g+w project.txt
$ chmod o+w project.txt
$ ls -l project.txt
-rwxrw-rw- 1 user staff 0 Sep 29 14:10 project.txt
