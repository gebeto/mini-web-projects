Sometimes we navigate around so much that we can lose track of where we are. Luckily, there's a command to print the working directory.

$  ls dir pwd  
/Users/user/Documents

Ah, there we are! pwd stands for "print working directory" - the address, or path, of the directory we're in.

Psst: on your computer, user will be your user name.

Might the command we're looking for be an abbreviation of the command's name?
