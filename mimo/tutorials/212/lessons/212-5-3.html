Whenever we run a program on a computer, so-called processes that belong to the program will execute in the background.

$  ps ps x  12300 ttys000 0:00.39 -bash PID    TT  STAT TIME    COMMAND
12301   ?? S   12:34:56 /Applications/Safari.app 
12300 ttys000 0:00.39   -bash
...

See that? If we just use ps, we're presented with the processes of the current session. ps x display all of the active processes.
