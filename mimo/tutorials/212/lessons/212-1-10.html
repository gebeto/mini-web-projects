When directories or files become obsolete, we can use rmdir or rm to get rid of them.

$ cd Documents
$ ls
readme.txt
$ mkdir Trash
$ touch Trash/trashme.txt
$ rm Trash/trashme.txt rmdir -rf Trash
$ ls
Trash
readme.txt readme.txt

Poof! We can use rmdir to remove directories and rm to remove files. If a directory isn't empty, we need to add an -rf option so rmdir includes its contents.

Psst: we have to be careful when we remove things because there's often no way to recover them.
