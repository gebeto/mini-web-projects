Let's explore manipulation in practice!

Create two text files and a Container directory. Then copy the files into Container, hop into it, and rename them. Finally, list only text files in the directory.

$ touch file1.txt
$ touch file2.txt
$ mkdir Container
$ cp file1.txt Container
$ cp file2.txt Container
$ cd Container
$ mv file1.txt readme1.txt
$ mv file2.txt readme2.txt
$ ls *.txt

Very impressive! You're really flexing your Terminal muscles.
