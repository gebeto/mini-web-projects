Of course, we can also make a list of friends.

friends = []
friends.append("Cece")
friends.append("Jess")
friends.append("Nick")
print(friends[ 0 1 2])
Cece Jess Nick

Do you see the difference? Python starts counting at 0, so the first item in the list has the position 0, the second item has the position 1, and so forth.
