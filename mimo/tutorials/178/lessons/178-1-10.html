It's project time! This time, however, let's use an HTML file with a <script> tag in the body section.

Create a chapter variable, assign 0 to it and increase it by 2. Add a sentence to the project variable that includes chapter and display it on the document.

<!doctype html>
<html>
 <body>
  <script>
   var project = "I think this is the beginning of a beautiful project";

   var chapter = 0;
   chapter += 2;

   project += " and this is only chapter " + chapter + "!";
   document.write(project);
  </script>
 </body>
</html>
I think this is the beginning of a beautiful project and this is only chapter 2!

Whee, we can place scripts within the <script> tag! Let's keep this file and use it throughout the course, okay?
