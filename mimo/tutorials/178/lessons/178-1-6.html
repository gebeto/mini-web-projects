So what are numbers good for? For example, we can use them for arithmetic operations.

var r = 4  + - * / 2;
var pi = 3.141592653589793;
console.log(2 + r * pi);

See that? We can use +, -, * and /, which we call arithmetic operators, to perform these, well, operations.
