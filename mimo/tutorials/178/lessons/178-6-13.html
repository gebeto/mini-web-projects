After we've talked about arrays in that much detail, why don't we get back to your project?

Create a days array and populate it, starting with "Sunday". Create a day variable that gets the current day from days. Finally, integrate the day in the sentence.

var date = new Date();
var dayNumber = date.getDay();
switch (dayNumber) {
 case 0: // Sunday
 case 6: // Saturday
  date.setHours(10);
  break;
 default:
  date.setHours(7);
}
date.setMinutes(0);
date.setSeconds(0);

var days = ["Sunday", "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday"];
var day = days[dayNumber];
console.log("It's " + day + ", " + date.toTimeString() + ". Time to get up.");
It's Monday, 07:00:00 GMT. Time to get up.

Sweet! We can dynamically get the current day with dayNumber, the variable that has the return value of date.getDay().
