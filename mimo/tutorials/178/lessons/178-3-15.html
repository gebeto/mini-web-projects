Let's make this mess a switch statement!

var season = "Summer";switch (season) {  
 case "Summer":  console.log("Hot season");  break; default:
  console.log("Other season");
}

Sweet! We create season and open the switch statement. We start with the case, display the description and add the break keyword. Finally, we add the default case.
