In this project addition, we'll have a look at your morning routine.

Create a day variable and give it a value. Use a conditional statement to check if it's a weekday and display the time to get up.

var day = "Monday";
switch (day) {
 case "Saturday":
 case "Sunday":
  console.log("It's 10 am. Time to get up.");
  break;
 default:
  console.log("It's 7 am. Time to get up.");
}
It's 7 am. Time to get up.

What a great use case for a switch statement! An if statement would require a lot of else if sections or logical operators.
