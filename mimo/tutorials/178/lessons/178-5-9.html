In this project, let's extend the morning routine from earlier on!

Create a date variable with the current date and time. Also, create a day variable with the day of date. Change the switch statement to use day.

var date = new Date();
var day = date.getDay();
switch (day) {
 case 0: // Sunday
 case 6: // Saturday
  date.setHours(10);
  break;
 default:
  date.setHours(7);
}
date.setMinutes(0);
date.setSeconds(0);
console.log("It's " + date.toTimeString()) + ". Time to get up.";
It's 07:00:00 GMT. Time to get up.
